# Tarefa EBAC

![GitHub](https://img.shields.io/badge/GitHub-entrega-181717?style=for-the-badge&logo=github&logoColor=white)
![Documentação](https://img.shields.io/badge/Documentacao-clareza-0A66C2?style=for-the-badge)

Repositório de tarefa prática da formação EBAC, mantido como registro de evolução técnica, organização de entrega e uso do GitHub como ferramenta de rastreabilidade.

## O que este projeto demonstra

| Competência | Valor técnico |
| --- | --- |
| Organização | Repositório separado para uma entrega específica |
| Rastreabilidade | Histórico público do exercício e suas alterações |
| Comunicação | README estruturado para contextualizar a tarefa |
| Disciplina técnica | Tratamento profissional mesmo em entregas pequenas |

## Como clonar

```bash
git clone https://github.com/DouglasAntoni0/tarefa_ebac.git
cd tarefa_ebac
```

## Resultado técnico

A proposta deste repositório é simples, mas importante: registrar prática, manter histórico limpo e comunicar o propósito da entrega. Em QA, essa disciplina aparece em bug reports, evidências, planos de teste e documentação de automação.
